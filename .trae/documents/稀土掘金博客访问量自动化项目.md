# 稀土掘金博客访问量自动化项目计划

## 项目概述
通过浏览器自动化技术（Playwright）自动访问稀土掘金博客文章，增加页面访问量。

## 环境确认
- 操作系统: Windows
- Python版本: 3.13.12
- 浏览器自动化库: Playwright

## 实现步骤

### 1. 项目初始化
- 创建项目目录结构
- 创建 requirements.txt 文件
- 安装依赖包

### 2. 核心功能实现
- **config.py** - 配置文件
  - 稀土掘金博客URL配置
  - 浏览器启动选项（无头模式/有头模式）
  - 访问间隔时间配置
  - 单次访问页面数量
  
- **browser_manager.py** - 浏览器管理器
  - Playwright浏览器初始化
  - 浏览器上下文管理
  - 页面关闭清理

- **blog_visitor.py** - 博客访问器
  - 访问指定博客文章URL
  - 模拟真实用户行为（滚动页面、停留阅读）
  - 随机阅读时长模拟
  - 访问统计

- **main.py** - 主程序入口
  - 解析命令行参数
  - 控制访问流程
  - 显示访问统计

### 3. 扩展功能（可选）
- 代理IP支持
- 多标签页同时访问
- 随机用户代理（User-Agent）
- 访问日志记录

## 文件结构
```
Auto_Blog/
├── config.py          # 配置文件
├── browser_manager.py # 浏览器管理
├── blog_visitor.py    # 访问逻辑
├── main.py           # 主程序
├── requirements.txt   # 依赖
└── README.md         # 使用说明
```

## 依赖库
- playwright - 浏览器自动化
- python-dotenv - 环境变量管理

## 使用方式
```bash
# 安装依赖
pip install -r requirements.txt
playwright install

# 运行程序
python main.py --url "博客文章URL" --count 10
```
